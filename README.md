# Telegram-бот для продажи товаров

Полнофункциональный асинхронный Telegram-бот для продажи товаров частным продавцом. Бот работает 24/7, автоматически отвечает на вопросы покупателей, показывает каталог товаров и помогает быстро связаться с продавцом.

## Возможности бота

### Для покупателей:
- **Каталог товаров** — просмотр товаров с пагинацией (по 5 товаров на странице)
- **Поиск** — поиск товаров по названию и описанию
- **FAQ** — ответы на часто задаваемые вопросы (доставка, оплата, возврат, контакты)

### Для продавца (админа):
- **➕ Добавление товаров** — пошаговый диалог добавления товара (название, описание, цена, фото)
- **Список товаров** — просмотр всех товаров с ID
- **Удаление товаров** — удаление товара по ID
- **Экспорт** — выгрузка всех товаров в JSON-файл

## ⚙️ Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
cd bot
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите:

```env
# Telegram Bot Token (получите у @BotFather)
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# Telegram ID администратора (продавца)
# Узнать свой ID можно через бота @userinfobot
ADMIN_ID=123456789

# Контакт продавца для связи
SELLER_CONTACT=@SellerName
```

### 3. Запуск бота

```bash
python bot.py
```

## Использование

### Покупатель:
1. Нажмите `/start` или кнопку «Запустить»
2. Выберите действие в меню:
   - **Каталог** — просмотреть все товары
   - **Поиск** — найти товар по ключевому слову
   - **FAQ** — узнать ответы на вопросы

3. В каталоге нажмите на товар для просмотра деталей
4. Свяжитесь с продавцом для подтверждения сделки

### Продавец (админ):
Доступны специальные команды:

- `/add` — добавить новый товар (пошаговый диалог)
- `/list` — показать все товары с ID
- `/delete <id>` — удалить товар по ID
- `/export` — экспортировать товары в JSON
- `/cancel` — отменить текущий диалог добавления

## 📝 Формат данных

### Товары (items.json)
```json
{
  "items": [
    {
      "id": 1,
      "title": "Наушники Sony WH-1000XM4",
      "description": "Беспроводные наушники с шумоподавлением",
      "price": 25000,
      "photo": null,
      "status": "available",
      "reserved_by": null,
      "reserved_at": null
    }
  ]
}
```

### FAQ (faq.json)
```json
{
  "questions": [
    {
      "id": "delivery",
      "question": "Доставка",
      "answer": "Доставка осуществляется по всей России..."
    }
  ]
}
```

## Настройки

В файле `config.py` можно изменить:

- `ITEMS_PER_PAGE` — количество товаров на странице каталога (по умолчанию 5)
